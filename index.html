⚙ INSTALACIÓN — HABILITAR CONTROL DE PC
01 INSTALAR PYTHON

Necesitás Python 3.10 o superior. Durante la instalación marcá la opción Add Python to PATH o no va a funcionar.

↗ DESCARGAR PYTHON
02 INSTALAR NGROK

ngrok crea un túnel para que esta web llegue a tu PC. Descargalo, descomprimilo y dejá el ngrok.exe en la misma carpeta que main.py.

↗ DESCARGAR NGROK

Registrate gratis en ngrok.com y corré ngrok config add-authtoken TU_TOKEN una sola vez.

03 DESCARGAR BACKEND

Descargá main.py y abrilo con un editor de texto. Reemplazá AQUI_LA_API_DE_GEMINI con tu API key gratuita de aistudio.google.com/app/apikey.

↓ DESCARGAR MAIN.PY
04 ARRANCAR CON UN CLICK

Descargá el iniciador y guardalo en la misma carpeta que main.py. Doble click → espera que se abran las ventanas → copiá la URL de ngrok → pegala abajo y hacé click en CONECTAR.

↓ DESCARGAR JARVIS.BAT

Solo Windows. En Mac/Linux corrés python main.py y ngrok http 8000 manualmente.

SIN CONECTAR
